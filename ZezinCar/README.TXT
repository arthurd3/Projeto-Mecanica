PROJETO ZEZIN CAR

TÍTULO DO PROJETO
Zezin Car - Sistema de Agendamento para Oficinas Mecânicas

OBJETIVO DO PROJETO
O objetivo deste projeto é criar um sistema simples e intuitivo para gerenciar uma oficina mecânica. O Zezin Car visa facilitar o agendamento de serviços, oferecendo uma interface amigável tanto para os clientes quanto para os funcionários da oficina. Com este sistema, os usuários poderão registrar clientes, mecânicos e boxes, além de realizar agendamentos de maneira eficiente.

COMO INICIAR O PROJETO
Para iniciar o projeto, execute o aplicativo. O sistema apresenta um menu com várias opções, permitindo que o usuário escolha a ação desejada. Para realizar um agendamento, é necessário que o cliente esteja registrado, assim como o mecânico e a box onde o serviço será realizado. As opções disponíveis incluem:

1. Cadastrar Serviço
2. Cadastrar Mecânico
3. Área do Cliente
4. Cadastrar Box
5. Realizar Agendamento
6. Consultar Disponibilidade de Boxes
7. Gerar Relatórios
8. Area Peças
0. Sair
